# DIEDIE
> 추가 예정

## 개요
장고를 이용한 게임 restful 서버입니다.

## 목표
게임 로그인 및 회원가입과 랭킹 DB 구현

## 사용 버전
- Python: 3.6.0
- Django: 2.0.7   

## 설치

1. [python3.6](https://www.python.org/downloads/) 이상이 설치되어 있어야합니다.
2. 가상환경 생성 및 활성화
```
$ python3 -m venv venv
$ source venv/bin/activate
```
3. requirements 설치
```
$ pip install -r requirements.txt
```
